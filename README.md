## 📚 项目简介

NeoTutorial 是一个基于 [VitePress](https://vitepress.dev/) 构建的 Neoforge 1.21.1 模组开发教程文档站点。本项目致力于为 Minecraft 模组开发者提供系统化、易理解的学习资源，帮助开发者从零开始掌握 Neoforge 模组开发技能。

## ✨ 主要特性

- ⚡️ **简单详细，适合新手入门** - 通过按主题循序渐进的方式，让读者逐步了解 Neoforge 的内容
- 🤔 **通俗易懂，带动读者思考** - 通过丰富的示例，让读者更轻松地理解 Neoforge 的各种功能特性  
- 📚 **可能是全网最全的 Neoforge 中文文档** - 还附带有专有名词对照表，让不懂英文的国人也能轻松阅读文档
